# Features

## Implemented

- variable assignment
- variable increment by one
- variable decrement by one
- repeat (count = result of an expression)
- compound expressions
- addition, subtraction, multiplication, division
- comments
- if
- else
- strings
- brackets
- logical comparisons (e.g. >=, < etc)
- user input
